HA.1.3 Create and solve a simple linear equation in context

                Illustration
                Karen and Keyla spent a total of $350 buying clothes at the mall. If Keyla spent $70
                more than Karen did, how many dollars did Keyla spend?
                Answer key
                If Karen spent x dollars, then Keyla spent (x + 70) dollars. The two spent a total of $350.
                Therefore, x + x + 70 = 350, or, 2x + 70 = 350. Subtracting 70 from both sides gives
                2x = 280, or, x = 140. Keyla spent 140 + 70 = $210.
